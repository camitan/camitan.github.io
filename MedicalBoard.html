Medical Board Web Application
Overview
User: PLTW students and teachers
Location | Time: Indianapolis | Summer 2019
Language/Library: HTML, CSS, jQuery, JavaScript
Tools: Adobe Illustrator


This is an intern project collaborated with designer Jordan Proffitt-Hilbert during summer 2019. It's a mocking medical board web application (deigned for ipad size), with tab style navigation, dark mdoe and light mode. Built with HTML, CSS, JSON, JavaScript, jQuery.

Goal for the project
  • Students can type in and view information on the app.
  • Make it visually interesting and look like a medical data examinator on iPad.
  • Have dark mode and light mode.
  • User-centered design for navigation, so users can click different tab easily.
Process


What was my role in the project?
Designer Jordan deigned the visual of the map and I'm making all the function happen as well as input my knowledge of frontend to discuss the options and limitations of the project.
Outcome (click to view)


What is the biggest challenge during this project?
I feel like I enjoyed so much building up this web application. Thanks to the designer Jordan the app is really colorful and cute, and it looks like a medical examinator iPad in a Sci-fi movie, I'm sure students will love it! The most interesting parts were making it look like a chrome-tab and user can switch color mode by light and dark. We also had a big discussion about whether or not we need a database to save users' inputs, so I contributed my knowledge in backend and explained my thoughts to designers. Due to the time and budget, eventually we did not pursue that road.
