Information Architecture Re-design Project
Overview
User: Golf Club Client
Location | Time: Bloomington, IN | Fall 2019
Type: UX design, Information structure, Wireframe
Tools: Axure, SiteMap, Excel


This is a individual project during the class Information Architecture at IU. This is a project to redesign a website that has structure problems by analyzing their web contents, navigation, structure and usability, sitemap and wireframe were created. Full report here: Full Report .

Goal for the project
  • To make the navigation more clear by relabeling
  • Users can locate the information quicker
  • The webiste structure should be more straightforward
  • To attract more people to join the club after re-design.
Process


Content Analysis
I collected all data from the webiste and analyze all of them line by line. Decide whether or not we keep/delete/merge the pages.


SiteMap
Restructure and design the sitemap


Wireframe
I created a new wireframe by Axure, adding notes to indicate what contents we need to insert into each section.


Outcome
Please review my full Wireframe at HERE .
What is the biggest challenge during this project?
To decide which page should belong to which category in the navigation is the most challenging park. There are so many ways to re-structure, but there are optimal ways I need to figure out to maximize the company's profits but also benefit to user experiences for the end users.
